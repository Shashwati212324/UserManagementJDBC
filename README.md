### User Management System - JDBC + MYSQL



MVC-like separation(Beans, DAO, GUI, Listeners)

This project is a desktop based User Management System built using Java - JDBC with database interaction.

It allows users to Register, Login, Select, Update and Delete records from a MySQL database through a simple graphical user interface.



## Tech Stack Used



1 Java (Core+ OOP)

2 JDBC (Database Connectivity)

3 MySQL Database

4 Eclipse IDE



## How the App works



1 User opens the application 

2 Chooses Login/Register

3 Login checks credentials

4 Register inserts new data 

5 User can edit , delete or update records

6 Logout when required
